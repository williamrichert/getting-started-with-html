Web Basics

World Wide Web

The Web is made up of a collection of HTML documents linked by hyperlinks. The web was invented in the early 1990's by Tim Berners-Lee and others. The Web is composed of computers that are optimized for running continuously to host and serve-up webpages (HTML documents). Called 'Servers' these computers receive requests sent to them by humans and their webbrowsers which send requests in the form of the protocol of the web, HTTP.

Hyper Text Markup Language (HTML)is the the main document type of the Web.

When people say cloud, they are really just talking about the internet.

HTTP Protocol

The internet uses HTTP to communicate with you, your browser, and servers. More about HTTP later...

HTML

HTML is made up of:

1) the text content (what you see) as well as
2) mark-up code that tells your browser what the text will look like or how it will be displayed.
3) references to other documents (e.g., videos or links to other pages).

In HTML, plain text is rendered as plain text. HTML markup tags start and end with angled brackets. Many tags have both an opening and a closing tag, sort of bracketing, the text on which they are applied. Tags without a closing tag are called void tags. The break tag is an example of a void tag. It's easy to forget to add the closing tag, but you will often see the result of this error show up on the webpage, and a good text editor, such as scratchpad.io will automatically create the end tag as soon as you write the closing angled bracket on the opening tag.

HTML Attributes - new concept

A tag attribute always equals some value, such as the url of a webpage or a link to an image. Examples would be: a href= www."udacity.com" Surround that with angled brackets, follow it with some text content and then an end tag and you will have a link. anchors are a's inside of angled brackets: they are for making links. Note, that the image or img tag is a void tag: img src= "url" alt= "txt" The alt= "some text here" gives the computers instructions on what to display to the user if the picture at the link is unavailable. The letters 'src' stand for source Do not forget to surround the url's with quotation marks. anchor tags use the href attribute while image tages use the src attribute

White space: White space gets ignored and all text is rendered continuously on one line unless you tell the computer otherwise.
Inline Elements vs. Block Elements: Inline elements occupy the same line if there is room. Block elements take up a whole line. Block Elements such as a p tag or form or a div tag create 'invisible boxes' that is, they create a new block of text.

Container elements: elements that do nothig but contain stuff inside themselves.
You can use div tags
to nest boxs of text
inside of other boxes by nesting the div tags. You can do the same with span-tags(span tags are inline elements, they demarcate boxes on one line).
You do not see these boxes in the browser - they are invisible unless otherwise styled into view - but you can know the insivible boxes are there demarcating text by how the previous sentences are broken up over several lines.
